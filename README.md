# 🚀 150 Hard AI Project Ideas — BTech

A modern, interactive web-based collection of **150 advanced AI/ML project ideas** designed for BTech students and developers.  
This project helps users explore high-quality, production-level AI ideas across multiple domains with an elegant UI.

---

## 📌 Features

- 🔍 **Search Functionality**
  - Quickly search project ideas by keywords, domains, or technologies

- 🧠 **150 Advanced AI Projects**
  - Carefully curated ideas across multiple domains

- 🎯 **Domain Filters**
  - Filter projects by categories:
    - Medical
    - Education
    - Agriculture
    - Smart City
    - Business
    - Cybersecurity
    - Environment
    - Legal
    - Social

- 📊 **Difficulty Levels**
  - Hard
  - Expert

- 🎨 **Modern UI Design**
  - Dark theme
  - Responsive layout
  - Smooth hover effects

---
